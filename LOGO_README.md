# BrisConnect Logo Instructions

To add your custom logo:

1. Add your logo image file to the `assets/` folder
2. Name it `logo.png` (or update the path in welcome_screen.dart if using a different name)
3. Supported formats: PNG, JPG, JPEG, GIF, WebP, BMP, WBMP
4. Recommended size: 200x200 pixels or larger (will be scaled down)

The app is now configured to load `assets/logo.png` as the main logo on the welcome screen.
